## 使い方
sample_aiの中にあるagent_learned.zipをダウンロードします。  
 [https://lunanaedu.github.io/puyopuyo-kai/]( https://lunanaedu.github.io/puyopuyo-kai/)を開きます。  
「ファイルを選択」をクリックし、ダウンロードしたagent_learned.zipを選択し、「load」をクリックします。
